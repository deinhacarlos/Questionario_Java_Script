Possibilidades com uso de JavaScript

Java Script e Java são a mesma coisa?

R: Não, Java e JavaScript não são a mesma coisa. Embora os nomes sejam semelhantes e muitas pessoas confundam as duas, elas são linguagens de programação distintas, com propósitos e características diferentes.

Quando surgiu o JavaScript?

R: O JavaScript surgiu no final dos anos 90, mais precisamente em 1995. Foi criado pela Netscape Communications Corporation, com o objetivo inicial de adicionar interatividade às páginas da web, tornando-as mais dinâmicas e responsivas.

Quem foi o responsável pela criação do JavaScript?

R: O criador oficial do JavaScript é Brendan Eich. Ele trabalhava na Netscape quando desenvolveu essa linguagem, que inicialmente se chamava Mocha, depois LiveScript e finalmente JavaScript.

Qual o caminho trilhado pelo reponsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?

R: Brendan Eich, o criador do JavaScript, trilhou um caminho bastante interessante após sua passagem pela Netscape. Embora o JavaScript tenha se tornado uma das linguagens de programação mais populares do mundo, a trajetória de Eich foi marcada por diversos projetos e empresas.
